Git Security Fix Project

This project demonstrates how to remove accidentally committed .env files from Git history.

What I Did

  • Found the commit that added the .env file
  • Removed .env from the entire Git history using git filter-branch
  • Added a .gitignore to prevent future accidents
  • Created a .env.example as a safe template
